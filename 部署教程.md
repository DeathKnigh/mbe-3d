# MBE 分子束外延 3D 交互平台 - 部署教程

## 📦 项目说明

这是一个基于 Three.js 的 **MBE（分子束外延）基础知识 3D 交互学习平台**，包含：

- **6 大模块**：MBE 原理、设备结构、工艺流程、材料体系、应用领域、优势局限
- **32 个核心知识点**：覆盖 MBE 技术全方位内容
- **3D 可视化**：MBE 设备模型 + 知识点环形展示
- **交互功能**：点击学习、模块筛选、进度跟踪
- **学习进度**：自动记录已学知识点

## 🚀 快速部署（3 种方案）

### 方案一：Vercel（⭐ 最推荐，1 分钟搞定）

**优点**：一键部署、免费 HTTPS、自动 CDN、支持自定义域名

#### 步骤：

1. **准备 GitHub 账号**
   ```bash
   # 如果没有 GitHub 账号，访问 https://github.com 注册
   ```

2. **上传代码到 GitHub**
   ```bash
   cd ~/Desktop/MBE-3D
   git init
   git add .
   git commit -m "MBE 3D 交互学习平台"
   git branch -M main
   git remote add origin https://github.com/你的用户名/mbe-3d.git
   git push -u origin main
   ```

3. **部署到 Vercel**
   - 访问 https://vercel.com
   - 用 GitHub 账号登录
   - 点击 "New Project"
   - 导入 `mbe-3d` 仓库
   - 点击 "Deploy"
   - 等待 1-2 分钟，完成！

4. **访问地址**
   - Vercel 会生成 `https://xxx.vercel.app` 域名
   - 可直接分享使用

---

### 方案二：Netlify（拖拽部署，最简单）

**优点**：无需 Git、拖拽上传、免费 HTTPS

#### 步骤：

1. 访问 https://www.netlify.com
2. 注册账号（可用 GitHub 登录）
3. 登录后点击 "Sites"
4. 把 `~/Desktop/MBE-3D` 整个文件夹拖到部署区域
5. 等待上传完成（约 30 秒）
6. 获得 `https://xxx.netlify.app` 访问地址

---

### 方案三：国内云服务器（阿里云/腾讯云）

**优点**：国内访问快、完全可控、可绑定域名

**成本**：约 100 元/年（入门配置）

#### 步骤：

1. **购买云服务器**
   - 阿里云：https://www.aliyun.com
   - 腾讯云：https://cloud.tencent.com
   - 配置：1 核 2G 足够（静态页面很轻量）
   - 系统：Ubuntu 20.04 或 22.04

2. **连接服务器**
   ```bash
   ssh root@你的服务器 IP
   ```

3. **安装 Nginx**
   ```bash
   apt update
   apt install nginx -y
   systemctl start nginx
   ```

4. **上传文件**
   
   **方法 A：scp 命令**
   ```bash
   # 在本地电脑执行
   scp -r ~/Desktop/MBE-3D root@服务器 IP:/var/www/html/
   ```
   
   **方法 B：FileZilla 图形化工具**
   - 下载安装 FileZilla
   - 主机：服务器 IP
   - 用户名：root
   - 密码：服务器密码
   - 端口：22
   - 远程目录：`/var/www/html/MBE-3D`

5. **配置 Nginx**
   ```bash
   nano /etc/nginx/sites-available/default
   ```
   
   修改为：
   ```nginx
   server {
       listen 80;
       server_name _;
       root /var/www/html/MBE-3D;
       index index.html;
       
       location / {
           try_files $uri $uri/ /index.html;
       }
   }
   ```

6. **重启 Nginx**
   ```bash
   systemctl restart nginx
   ```

7. **配置防火墙（如有）**
   ```bash
   ufw allow 80/tcp
   ufw reload
   ```

8. **访问测试**
   - 浏览器打开 `http://你的服务器 IP/MBE-3D/`

9. **（可选）绑定域名**
   - 购买域名（约 50 元/年）
   - 添加 A 记录指向服务器 IP
   - 修改 Nginx `server_name` 为域名
   - 用 Certbot 配置 HTTPS（免费）

## 🆘 常见问题

### Q: 页面加载后显示空白？
A: 打开浏览器开发者工具（F12），查看 Console 是否有错误。通常是文件路径问题。

### Q: 模块点击没反应？
A: 确保使用的是现代浏览器（Chrome、Firefox、Edge、Safari），Three.js 需要 WebGL 支持。

### Q: 如何更新内容？
- **Vercel/Netlify**：修改代码后推送到 GitHub 自动更新
- **云服务器**：重新上传文件，刷新浏览器（Ctrl+F5 强制刷新）

### Q: 可以离线使用吗？
A: 可以！下载 Three.js 本地文件，修改 HTML 中的引用路径即可完全离线运行。

### Q: 如何下载 Three.js 本地文件？
```bash
cd ~/Desktop/MBE-3D
mkdir lib
curl -o lib/three.min.js https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js
```
然后修改 `index.html`：
```html
<script src="lib/three.min.js"></script>
```

## 📊 性能优化建议

1. **启用 Gzip 压缩**（Nginx）
   ```nginx
   gzip on;
   gzip_types text/html text/css application/javascript;
   ```

2. **添加缓存头**
   ```nginx
   location ~* \.(js|css|png|jpg)$ {
       expires 30d;
       add_header Cache-Control "public, immutable";
   }
   ```

3. **使用 CDN**
   - 静态资源托管到 CDN 加速访问
